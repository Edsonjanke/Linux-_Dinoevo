3.2 Chave Seletora de Eixo (X / OFF / Z)

  CHAVE ROTATIVA 3 POSICOES           DB15 MACHO
  (1 polo, 3 posicoes)                (soldar)
  ┌─────────────┐
  │             │
  │ COM ────────┼──────────────────── Pino 14 (COM/GND)
  │             │
  │ Pos 1 (X)──┼──────────────────── Pino 11 (Eixo X)
  │ Pos 2 (OFF)┼──── nao conecta
  │ Pos 3 (Z)──┼──────────────────── Pino 7  (Eixo Z)
  │             │
  └─────────────┘

  Funcionamento:
    Posicao X  : pino 11 conecta ao COM (GND) → jog eixo X
    Posicao OFF: nenhum pino conecta → jog desabilitado
    Posicao Z  : pino 7 conecta ao COM (GND) → jog eixo Z

3.3 Chave Seletora de Escala (x1 / x10 / x100)

  CHAVE ROTATIVA 3 POSICOES           DB15 MACHO
  (1 polo, 3 posicoes)                (soldar)
  ┌─────────────┐
  │             │
  │ COM ────────┼──────────────────── Pino 14 (COM/GND)
  │             │
  │ Pos 1 (x1) ┼──────────────────── Pino 2  (Escala x1)
  │ Pos 2 (x10)┼──────────────────── Pino 5  (Escala x10)
  │ Pos 3(x100)┼──────────────────── Pino 9  (Escala x100)*
  │             │
  └─────────────┘

  Valores por click do encoder em cada escala:
    x1   = 0.025 mm   (desbaste / movimentacao rapida)
    x10  = 0.0025 mm  (acabamento)
    x100 = 0.00025 mm (ultra-fino)

  * Pino 9 (x100): GPIO a confirmar com readhmid.
    Se nao funcionar, usar apenas x1 e x10.

5. FIACAO DA PLACA 7i92 - OUTROS CONECTORES

5.1 Spindle (Placa de Potencia / VFD / Inversor)

  PLACA 7i92                              VFD / INVERSOR
  ┌──────────┐                           ┌──────────────┐
  │          │                           │              │
  │  PWM0  + ├───────────────────────────┤ Entrada AIN  │
  │        - ├───────────────────────────┤ GND          │
  │          │  (sinal 0-5V proporcional │              │
  │          │   ao RPM: 0V=0, 5V=2000) │              │
  │          │                           │              │
  │ GPIO.013 ├───────────────────────────┤ Entrada FWD  │
  │      GND ├───────────────────────────┤ COM          │
  │          │  (HIGH = gira horario M3) │              │
  │          │                           │              │
  └──────────┘                           └──────────────┘

  PWM0: Velocidade do spindle (duty cycle proporcional)
    - Frequencia: 20kHz (ajustar se VFD nao responder)
    - Escala: 2000 RPM = 100% duty
    - Se o VFD precisa 0-10V, usar divisor/amplificador

  GPIO.013: Direcao CW (saida digital)
    - HIGH = M3 (horario)
    - LOW = parado ou M4

5.3 E-Stop (Emergencia Principal)

  PLACA 7i92                     BOTAO E-STOP
  ┌──────────┐                  ┌──────────────┐
  │          │                  │   COGUMELO   │
  │ GPIO.019 ├──────────────────┤ NF (contato  │
  │      GND ├──────────────────┤  normalmente │
  │          │                  │   fechado)   │
  └──────────┘                  └──────────────┘

  FUNCIONAMENTO:
    Normal:     GPIO.019 = HIGH (circuito NF fechado)
    Emergencia: GPIO.019 = LOW  (circuito aberto)
    A maquina PARA imediatamente.

  ATENCAO: SEMPRE usar botao NF (normalmente fechado)!
  Se o fio romper, a maquina para (fail-safe).

10. NOTAS

PWM FREQUENCY: Atualmente 20kHz. Se o VFD nao responder, mudar para 500Hz no Dino_Evo.hal linha 10:
setp hm2_7i92.0.pwmgen.pwm_frequency 500
SPINDLE SCALE: Atualmente 2000 RPM = 100% duty.
Se o RPM maximo real for diferente, ajustar no Dino_Evo.hal linha 218:
setp hm2_7i92.0.pwmgen.00.scale XXXX
ENCODER SPINDLE: Scale = 400 (pulsos por volta).
Se o encoder for diferente, ajustar no Dino_Evo.ini [SPINDLE_0]:
ENCODER_SCALE = XXX
ESCALA x100 (pino 9): GPIO nao confirmado. Rodar na maquina:
sudo mesaflash --device 7i92 --addr 10.10.10.10 --readhmid
para descobrir qual GPIO corresponde ao pino 9 do DB15.
